Tiny Python is a minimalist implementation of Python in 64K. 
It was originally written by Phil Hassey.

I, Prasanth Madhavan, Sumith S., and Sunil Kumar tried to 
understand the working of TinyPython Virtual Machine v1.1. 
The study report is provided in this repo.

"More" information can be found in:
http://www.philhassey.com/blog/

TinyPy v1.1 can be downloaded from:
http://code.google.com/p/tinypy/downloads/list
